## Accesso al registro

Il registro è raggiungibile all'indirizzo:

> **https://registro.efor.it**

Le credenziali (username + password iniziale) vengono comunicate dall'istituto via email all'inizio dell'anno scolastico o quando l'account viene attivato.

Username: in genere `cognome.iniziale_nome` (es. `bianchi.m` per Mario Bianchi).
Email associata: solo per ricezione comunicazioni e recupero password, **non si usa per il login**.

## Glossario condiviso

Termini ricorrenti in tutti i manuali:

| Termine | Significato |
|---|---|
| **Cattedra** | Assegnazione `docente + materia + classe` che abilita il docente a operare su quella classe per quella materia. |
| **Sede** | Una delle sedi dell'istituto. Le festività e gli orari possono essere specifici per sede. |
| **Scansione oraria** | Definizione delle ore standard di lezione (1ª 8:00-9:00, 2ª 9:00-10:00, ...) per ogni giorno della settimana. |
| **Orario** | Calendario settimanale che assegna ogni cattedra a uno slot `giorno × ora`. |
| **Lezione** | Singola istanza di una materia in una determinata data e ora, firmata dal docente. |
| **Firma** | Operazione con cui il docente certifica la sua presenza in aula per un'ora di lezione. |
| **Compresenza** | Più docenti firmano la stessa ora (es. ITP + curricolare, sostegno + curricolare). |
| **Sostituzione** | Lezione tenuta da un docente diverso dal titolare della cattedra. |
| **Modulo formativo** | Percorso didattico identificato (es. moduli PCTO, moduli interdisciplinari). |
| **Scrutinio** | Periodo di valutazione (intermedio, finale, integrativo). |
| **Periodo** | Suddivisione dell'anno scolastico (es. trimestre, quadrimestre). |
| **Coordinatore** | Docente con incarico di gestione di una specifica classe. |
| **Segretario** | Docente che redige il verbale del consiglio di classe. |
| **Staff** | Sottoinsieme dei docenti con ruoli amministrativi elevati (collaboratori del Dirigente). |
| **Alias** | Funzione che permette agli amministratori di "vedere" il registro come un altro utente, per supporto e verifica. |
| **BES** | Bisogni Educativi Speciali. |
| **PCTO** | Percorsi per le Competenze Trasversali e l'Orientamento (ex Alternanza). |
| **RSPP** | Responsabile Servizio Prevenzione e Protezione (sicurezza). |
| **DSA** | Disturbi Specifici dell'Apprendimento. |

## Convenzioni di lettura

- **Percorsi** nei menu sono scritti come `Sezione → Sottosezione → Voce`.
- **URL** delle pagine sono indicati come `/percorso/sotto-percorso`.
- I **pulsanti** sono in grassetto: **"Salva"**, **"Aggiungi"**.
- I **campi obbligatori** sono marcati con `*` come nell'interfaccia.
- I **rimedi rapidi** ai problemi più frequenti sono nella sezione "FAQ / Errori comuni" di ogni manuale.

## Recupero password

Se hai dimenticato la password:

1. Vai su https://registro.efor.it/login
2. Click su **"Hai dimenticato la password?"**
3. Inserisci la tua **email** registrata
4. Riceverai una nuova password temporanea per email
5. Al primo accesso ti verrà chiesto di cambiarla

Se non ricordi nemmeno l'email, contatta la segreteria dell'istituto.

## Supporto

Per problemi tecnici contattare:
- la **segreteria** dell'istituto per problemi di account, anagrafica, dati
- l'**amministratore di sistema** (referente IT) per malfunzionamenti del registro
